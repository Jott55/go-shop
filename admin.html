How to run postgres server in Linux

1. Download podman or docker for your distribution

2. Run the following code

Before running the container run the following command (defining container variables)

export yourPassword=1234
export yourUser=$USER

you can change the values for your needs

In Docker

sudo docker run --name shop-postgres -e "POSTGRES_PASSWORD=$yourPassword" -e "POSTGRES_USER=$yourUser" -d postgres

In Podman

podman run --name shop-postgres -e "POSTGRES_PASSWORD=$yourPassword" -e "POSTGRES_USER=$yourUser" -d docker.io/postgres

3. Than use should have a container image of postgres running in background

so your user and database name should be $yourUser -- meaning the variable you defined using export yourUser=$USER -- If not modified the name should be your desktop username, that appears when entering this command `echo $USER`

the default password is 1234

the host is localhost

and the default port for postgres is 5432

What if my container stoped?

my recommendation to verify errors is searching for logs

you should read the docker or podman documentation but if it just stopped cuz you restarted your pc it you could start it again using

docker start shop-postgres
podman start shop-postgres
